# Обновление системы комментариев - Краткая справка

## Что изменилось

### ✅ Новые функции
1. **Вкладка "Ответы админа"** - отдельный просмотр всех ответов администраторов
2. **Фильтр по упражнениям** - быстрая навигация по комментариям к конкретным упражнениям
3. **Звездочка для важных вопросов** - пометка важных Q&A, которые показываются в топе
4. **Улучшенная сортировка** - важные комментарии и личные ответы админа в приоритете

### 📝 Измененные файлы
**Backend:**
- `src/models/Comment.model.ts` - добавлено поле `starred`
- `src/routes/comment-admin.routes.ts` - новые endpoints и фильтры
- `src/routes/comment.routes.ts` - обновлена сортировка

**Admin Panel:**
- `admin-panel/src/pages/Comments.tsx` - новые вкладки, фильтры и UI

## Использование в админке

### Вкладки
- **Общий поток** - все комментарии для модерации (как раньше)
- **Ответы админа** - только ответы администраторов (новое)

### Фильтры
- Поиск по тексту
- Статус (На модерации, Одобренные и т.д.) - только в "Общем потоке"
- Приоритет (Обычные, Горящие) - только в "Общем потоке"
- **Упражнения** - на обеих вкладках
- Ждущие ответа - только в "Общем потоке"

### Звездочка
1. Откройте комментарий (клик по нему в списке)
2. Нажмите кнопку "⭐ Сделать важным"
3. Важный комментарий будет показываться в топе в приложении

## Поведение для пользователей

Комментарии в приложении сортируются так:
1. ⭐ Важные (starred) комментарии
2. 💬 Личные ответы админа текущему пользователю
3. 📅 Все остальные (новые первые)

## API Changes

### Новые endpoints
- `GET /api/admin/comments?view=admin-replies` - получить ответы админа
- `PUT /api/admin/comments/:id/starred` - пометить как важное
- `GET /api/admin/comments/exercises/list` - список упражнений

### Обновленные endpoints
- `GET /api/admin/comments?exerciseId=...` - фильтр по упражнению
- `GET /api/comments` - обновленная сортировка

## Troubleshooting

**Проблема:** Вкладка "Ответы админа" пустая  
**Решение:** Проверьте, что вы отвечали на комментарии (они должны иметь `respondedBy`)

**Проблема:** Звездочка не сохраняется  
**Решение:** Проверьте, что миграция выполнена и поле `starred` добавлено в БД

**Проблема:** Фильтр по упражнениям не работает  
**Решение:** Убедитесь, что endpoint `/api/admin/comments/exercises/list` доступен
