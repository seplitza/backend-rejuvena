# Исправление админ-панели - 02.04.2026

## Проблема
Админ-панель по адресу `api-rejuvena.duckdns.org/admin/` не загружалась. 
В консоли браузера ошибки 404 на файлы `main.tsx.js` и `vite.svg`.

## Причина
На сервере лежала устаревшая версия сборки админки с неактуальными именами файлов.

## Решение

### 1. Исправлена nginx конфигурация
В файле `/etc/nginx/sites-available/rejuvena` исправлена директива `try_files`:

```nginx
location /admin/ {
    alias /var/www/rejuvena-backend/admin-panel/;
    try_files $uri $uri/ /admin/index.html;  # Было: /index.html =404
    index index.html;
}
```

### 2. Пересобрана и задеплоена админка
```bash
cd /var/www/rejuvena-backend/Backend-rejuvena
./deploy-admin.sh
```

Текущие файлы сборки:
- `dist/assets/index-BGM7gsJ7.js` (1.8 MB)
- `dist/assets/index-B6JUJVKp.css` (31.5 KB)
- `dist/index.html`
- `dist/vite.svg`

### 3. Удален конфликтующий backup-файл
```bash
rm -f /etc/nginx/sites-enabled/rejuvena.backup
systemctl reload nginx
```

## Результат
✅ Админ-панель доступна по адресу: https://api-rejuvena.duckdns.org/admin/
✅ Все статические файлы загружаются корректно
✅ Nginx конфигурация оптимизирована
